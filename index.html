Mount Doom

Unity, Mobile, Idle Game, Online

An idle game where you collect a variety of heroes. You can send them on missions or trade them with other players online.
One focus of this project was the implementation of an Mobile Idle game that plays in realtime while the app is closed.
Another focus was the server backend to allow for player to player trades (and you will probably never guess how I solved that feature).

What would Satan do?

Unity, Mobile, AR, Bluetooth, Digitale Boardgame

In this mysterious social deduction game a witch coven gathered to figure out which of them is possessed by Satan. The phones of the witches project the game pieces into AR-Space and serve as their device to secretely vote against each other. Will they find the culprit before it is too late?
The phones are connected via bluetooth, which was maybe not the most efficient solution, but a good learning opportunity.
